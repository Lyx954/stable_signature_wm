## 3.5 AIGC 检测模块实现——数字水印主动检测（Stable Signature）

### 3.5.1 环境与依赖
- Python 3.10+
- torch
- torchvision
- numpy
- Pillow
- flask（本地 Web 可视化）
- ffmpeg / ffprobe（视频抽帧，建议安装）

### 3.5.2 API 输出格式与内容示例
**单图检测（核心字段）**
```json
{
  "filename": "photo.jpg",
  "file_type": "image",
  "has_watermark": true,
  "confidence": 0.98,
  "bit_accuracy": 0.99,
  "word_accuracy": 1.0,
  "extracted_bits": "111010110101000001010111010011010100010000100111",
  "expected_key": "111010110101000001010111010011010100010000100111",
  "dimensions": "512 x 341",
  "size_kb": 158.0,
  "error": null
}
```

**视频检测（抽帧）**
```json
{
  "video_path": "clip.mp4",
  "has_watermark": true,
  "frames_checked": 5,
  "frames_with_watermark": 4,
  "consensus_bits": "111010110101000001010111010011010100010000100111",
  "consensus_accuracy": 0.93,
  "confidence": 0.91,
  "frame_results": [
    { "image_path": "frame_000.png", "bit_accuracy": 0.98, "confidence": 0.96 }
  ]
}
```

### 3.5.3 怎么实现的
1. 采用 Meta 的 Stable Signature（facebookresearch/stable_signature）方案，仅保留 **检测/提取**，去除嵌入与攻击测试功能。  
2. 使用 48-bit msg_decoder（torchscript 约 1.2MB）进行推理，GPU 非必需。  
3. 图像检测流程：  
   - 读取图片 → resize/crop → normalize  
   - msg_decoder 输出 48-bit → 与 key_str 比对  
   - 计算 bit_accuracy / word_accuracy / confidence  
4. 视频检测流程：  
   - ffmpeg 抽帧（默认 5 帧）  
   - 每帧调用图像检测  
   - 汇总 frames_with_watermark + 计算 consensus_bits  
5. 接入 SourceTrust：  
   - 复制 integration/ 目录文件到 backend/app/ 对应位置  
   - main.py 注册路由  
   - 服务入口：ai_watermark_detection_service → watermark_detection_service
